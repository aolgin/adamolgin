Social Media Poster

Complete

This application makes it easy to post natively across multiple social media platforms simultaneously. While there are exsiting integrations between the platforms, using them can impact the maximum visibility of a post. The algorithms used by each platform tend to prioritize posts which were created natively on that platform rather than those shared to it from another platform. Posts made via this app will be recreated manually on each platform and therefore will likely receive more views than they would had they been shared across platforms. Created as a group project in my Software Development course at Northeastern University.

FAT File System Implementation

Complete

One of four major projects for my Systems & Networks course, worked on with my colleague. This particular project required us to write a usable FAT file system using C and the FUSE library. The end result can be mounted, written to, and read from. Sadly, we lacked the time to attempt to add in multi-level directory support, but I suppose you could simulate that in a sense with a single-level implementation such as ours.

Transport Protocol

Complete

One of four major projects for my Systems & Networks course, worked on with my colleague. This particular project required us to devise our own transport protocol, using TCP as a reference. Our protocol is a version of TCP that uses a static window, rather than an ever changing one. We currently have not implemented in some of the features of popular TCP versions, such as a congestion window, the two stages of TCP, and some more. However, ours still works well, even if it is not the best performing version of TCP out there.
